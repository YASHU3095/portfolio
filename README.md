## ✨ Features

- 🎨 **Dark Premium Design** — Deep dark theme with purple/cyan gradient accents
- ✨ **Animated Particle Background** — Interactive canvas with connected floating particles
- 🖱️ **Custom Cursor** — Smooth dual-layer cursor with hover effects
- ⌨️ **Typed Text Animation** — Role titles cycle automatically with typewriter effect
- 🌀 **Orbital Profile Rings** — Spinning animated rings around the profile photo
- 📊 **Animated Skill Bars** — Progress bars that animate on scroll into view
- 🃏 **Staggered Project Cards** — Cards reveal with cascade animation on scroll
- 📱 **Fully Responsive** — Optimized for desktop, tablet, and mobile
- 🔗 **Smooth Scrolling** — Buttery navigation between all sections
- 📬 **Contact Form** — With success feedback animation

## 🚀 Run Locally

```bash
# Clone the repository
git clone https://github.com/YASHU3095/portfolio.git

# Navigate to the project
cd portfolio

# Start the dev server
npm start
```

Then open **[http://localhost:3000](http://localhost:3000)** in your browser.

> **No build step needed** — it's pure HTML/CSS/JS served by `npx serve`.
